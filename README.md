# Radical Red Pokedex

The non-official Pokedex for Radical Red, a Pokemon Fire Red ROM hack.

## Under Construction 🚧

Please note that this Pokedex is still under construction.

## About

This Pokedex is a project that is inspired from https://dex.radicalred.net but aims to provide a more modern and user-friendly experience.

## Features

These features are currently available on https://dex.radicalred.net but will be expanded upon in this project.

- Quick search for Pokemon by name
- Review Pokemon stats, abilities, moves, and evolutions, and more
- Filter Pokemon by type
- Filter Pokemon by moves
- Filter Pokemon by abilities
- More to come as the project progresses
